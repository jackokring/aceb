About the App

The app is a demonstration of a framework for interacting with droid. A simple OSAdapter is created between the coding of the demonstration and various machines. A WebView is used, and so JavaScript can use the OSAdapter object to call on droid functions. There is also a method release() which must be called after using any method except machine() which itself send a text command to the other possibility.

There is also a Machine interface describing a simple 16-bit architecture, with the same, excepting machine() but including setMachine(), Java interface. This class extension of Tester is AceB. There may be other extensions added. The flexibility comes from the WebView launch first, but other components of the simple OS in the OSAdapter interface exist. There is an EditView for entry, and presentation before entry, of text. There is also a DisplayTerminal with a similar idea to the J2ME GameCanvas but nothing beyond simple essentials. There is also Audio and Joy accelerometer functionality available in the simple OS. More functionality may be supplied by extentions of the OSAdapter interface, as that interface and Machine are now set in design functionality.

Passing information back from a Machine to a JavaScript can be achieved by outURL(“javascript:<here>”) in the Machine interface. The display will then switch back to the WebView. The seeing of the interprocess communication in the EditView is seen as a good thing, although if the time to process is fast, it may not be displayed, or you may not be fast enough to read it. Two Machine instances can communicate in a similar way.

There is also a facility to share Machine instances, but the JavaScript secondary machine is not programable or sharable in the same way. May be some one will develop a Machine which can do this by downloading the current JavaScript into a Machine instance. There is also a facility to search for words a Machine understands, and to view any help available, and some settings to help with sensible defaults.

Eventually there may be a Hall of Fame feature, where good examples of shared Machine instances are bundled with the .apk file, or on the market. Hope you get to enjoy.
